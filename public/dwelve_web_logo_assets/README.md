# Dwelve Website Logo Assets

Generated website logo package based on the attached Dwelve logo reference and the color system in `design-system.md`.

Canonical repo references:

- `docs/design-system.md` contains the design rules, logo color rules, typography rules, and implementation mapping.
- `docs/brand-assets.md` contains the maintained asset inventory, served URLs, and usage guidance.
- Files under `public/dwelve_web_logo_assets/docs/` are generated package notes and should not replace the repo docs above.

## Included

- `logos/` — SVG logo variants for light mode, dark mode, icon-only, wordmark, stacked, and monochrome use.
- `png/` — transparent PNG exports at 512, 1024, and 2048 px.
- `favicon/` — favicon `.ico`, SVG, and PNG sizes.
- `app-icons/` — Apple and PWA icons.
- `social/` — Open Graph and Twitter/X card preview images.
- `docs/design-system.updated.md` — updated design-system file with relevant color/style/logo wording changes.
- `docs/dwelve-brand-tokens.css` — small CSS helper containing core brand variables.

## Brand colors

- Ink: `#0F1430`
- Violet: `#7B61FF`
- Violet Deep: `#6A4FF0`
- Violet Light: `#8E78FF`
- Mist: `#F3F4FF`
- Gradient: `linear-gradient(135deg, #8E78FF 0%, #6A4FF0 100%)`

## Notes

The SVGs are clean recreated vector-style files, not crops from the screenshot. The wordmark remains a bold geometric sans style to match the approved logo direction.
