# Dwelve Landing Page: Additional Content Suggestions

This file contains suggested content to add to the Dwelve landing page so it feels more specific, trustworthy, and product-focused.

The current landing page already explains the basic idea well:

> Create tests online, share them with students, and get instant grading.

The main issue is that the page still feels a bit generic. It should explain Dwelve's stronger product story more clearly:

- AI-assisted test creation
- PDF-to-test generation
- teacher review and control
- role-based usage for admins, teachers, and students
- private learning center and school positioning
- deeper analytics and reporting
- clearer trust and pricing content

---

## 1. Add a Clear “What Dwelve Does” Section

### Suggested Section Title

## Turn lessons, PDFs, and question banks into online tests

### Suggested Copy

Dwelve helps teachers create online tests from scratch, reuse old questions, or generate test drafts from learning materials. Teachers stay in control, review every question, then share the test with students by link or code.

### Why Add This?

The hero currently explains that users can create and take tests online, but it does not fully explain what makes Dwelve different. This section gives visitors a clearer product explanation early on.

---

## 2. Add AI/PDF Test Generation

This should become one of the strongest parts of the landing page.

### Suggested Section Title

## Create tests from PDFs with AI

### Suggested Copy

Upload lesson materials, worksheets, or notes, and Dwelve can help draft test questions for you. Review, edit, remove, or reorder questions before publishing the final test.

### Suggested Bullet Points

- Upload a PDF, worksheet, or lesson material
- Generate draft questions automatically
- Review and edit every question before publishing
- Save approved questions to your question bank
- Build tests faster without losing teacher control

### Important Wording

Use words like:

- draft
- suggest
- review
- approve
- teacher-controlled

Avoid saying:

- AI creates perfect tests
- fully automatic exams
- no teacher needed

AI should sound helpful, not like it is replacing the teacher. Nobody wants a robot confidently inventing exams and calling it education.

---

## 3. Add a “Teachers Stay in Control” Trust Section

### Suggested Section Title

## Teachers stay in control

### Suggested Copy

AI can help draft questions, but teachers approve everything before publishing. Edit wording, change answers, adjust time limits, and choose exactly which questions students receive.

### Suggested Bullet Points

- Review AI-generated questions before publishing
- Edit question text, answers, and options
- Remove weak or irrelevant questions
- Set time limits and test rules
- Publish only when the test is ready

### Why Add This?

Teachers may not trust AI if the page makes it sound like Dwelve automatically creates and sends tests without human review. This section makes the product feel safer and more professional.

---

## 4. Add Role-Based Content

Right now, the page mostly talks to teachers and students. If Dwelve is also for schools and learning centers, the landing page should explain each role.

### Suggested Section Title

## Built for every role in your learning center

### Suggested Cards

#### For admins

Manage teachers, classes, students, and test activity from one place.

#### For teachers

Create tests, reuse questions, set timers, and review results instantly.

#### For students

Take tests from any device without installing an app.

### Why Add This?

Different users visit the page with different concerns. Admins care about management, teachers care about creating and grading tests, and students care about ease of access.

---

## 5. Add School and Private Learning Center Positioning

### Suggested Section Title

## Made for schools and private learning centers

### Suggested Copy

Dwelve helps education teams replace paper tests with online exams, automatic grading, reusable question banks, and clear performance reports.

### Suggested Use Cases

- Weekly quizzes
- Placement tests
- Mock exams
- Homework checks
- Final exams
- Group progress tracking

### Why Add This?

The phrase “modern classrooms” sounds nice, but it is broad. Naming schools and private learning centers makes the target audience much clearer.

## 6. Add a Stronger Analytics Section

The current page mentions performance insights, but it should explain the analytics value more clearly.

### Suggested Section Title

## See what students understand, not just their scores

### Suggested Copy

After each test, Dwelve helps teachers understand class performance, difficult questions, top performers, and areas that need review.

### Suggested Bullet Points

- View average scores
- Check individual student results
- Find commonly missed questions
- Identify top performers
- Track class-level performance
- Review results immediately after submission

### Optional Features to Mention Only If Supported

- Export results
- Track progress over time
- Compare class performance across tests
- Download reports

Do not promise these unless they are actually supported or planned soon. Marketing lies are just bugs wearing perfume.

## 7. Add Product Preview Content

The dashboard mockup looks good, but the page should show more specific product moments.

### Suggested Product Preview Blocks

#### Test Editor

Create questions, set correct answers, choose test settings, and control the exam duration.

#### AI/PDF Generator

Upload learning material and generate a draft test that teachers can review before publishing.

#### Student Test Screen

Students open a link or code and complete the test directly in their browser.

#### Results Dashboard

Teachers see scores, submissions, missed questions, and class performance after the test.

#### Question Bank

Save useful questions and reuse them in future quizzes, homework, or exams.

## 8. Fix or Replace Trust Claims

The page currently uses trust elements such as:

- “Trusted by 500+ schools”
- university names like Stanford, MIT, Harvard, Oxford, and Yale

Only keep these if they are real and verifiable.

### Safer Alternatives

#### Option 1

## Built for modern schools and learning centers

#### Option 2

## Designed for teachers, tutors, and education teams

#### Option 3

## Built with feedback from real educators

### If You Have Real Early Data

Use honest numbers, even if they are small:

- Used by 120 students in beta
- Tested with local teachers
- Built for private learning centers
- Currently in early access

Real small proof is better than fake giant proof. Visitors can smell fake credibility faster than developers can create another unnecessary JavaScript framework.

## 11. Expand the FAQ

The current FAQ is useful but too short. Add questions that handle common doubts.

### Suggested FAQ Items

#### Can I generate tests from a PDF?

Yes. Teachers can upload learning material and generate draft questions, then review and edit them before publishing.

#### Can teachers edit AI-generated questions?

Yes. Teachers can change, delete, reorder, or approve questions before students take the test.

#### Is Dwelve for schools or private learning centers?

Dwelve is designed for teachers, schools, tutors, and learning centers that need simple online testing.

#### Can students take tests on phones?

Yes. Dwelve works in the browser on phones, tablets, and laptops.

#### Do students need an account?

Students can take tests based on the access method your school or teacher chooses.

> Adjust this answer based on the actual product flow.

#### Can admins manage multiple classes or teachers?

Yes, if your Dwelve plan supports admin and center management features.

> Only include this if the feature exists or is planned for release.

#### Can I reuse questions in future tests?

Yes. Teachers can save useful questions to a question bank and reuse them in future quizzes, homework, or exams.

#### Can I set a time limit?

Yes. Teachers can set a test duration, and submissions can close automatically when time runs out.
