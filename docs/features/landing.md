# Landing Pages

The landing route group contains the public marketing experience:

```txt
src/app/(landing)
```

## Rules

- Follow `docs/design/design-system.md`.
- Use translation keys for user-facing copy.
- Use approved brand assets from `public/logo` and `public/images`.
- Do not introduce competing fonts or ad-hoc colors.
- Keep marketing components route-local until they are reused elsewhere.
- Preserve responsive layout for English, Russian, and Uzbek Latin.
